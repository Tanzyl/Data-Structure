# Deployment Instructions for Tanzyl

## GitHub Setup Commands

After creating the repository on GitHub, run these commands:

```powershell
git remote add origin https://github.com/Tanzyl/data-structure-visualizer.git
git push -u origin main
```

**Note:** Replace `data-structure-visualizer` with your actual repository name if different.

## Enable GitHub Pages

1. Go to your repository on GitHub
2. Click on "Settings"
3. Scroll down to "Pages" section
4. Under "Source", select "main" branch
5. Click "Save"
6. Your site will be live at: `https://tanzyl.github.io/data-structure-visualizer/`
